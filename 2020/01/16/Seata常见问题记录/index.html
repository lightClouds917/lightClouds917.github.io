Seata常见问题记录

  • Seata
  • 2020-01-16

Seata大群中，有不少用户反馈问题，部分典型问题，已经整理为官方的faq,还有部分问题，未进入faq，这里做个记录，万一自己以后碰到，或者有社区成员遇到重复问题，可以有点帮助。

1.Could not found property service.disableGlobalTransacion.

Could not found property service.disableGlobalTransacion.try to use default value instead.

处理：这个输出是1.0的bug，只有file配置中心有问题，因为starter是通过代理类获取属性，而这个输出的地方是内部引用没走代理，导致获取不到，暂时可以在file.conf配置这个属性 。
